## Resume Builder

A comprehensive Flutter-based resume builder application with local storage and PDF generation capabilities.

### Location
`resume_builder/`

### Features
- Create and manage professional resumes
- Multiple sections: Personal Info, Work Experience, Education, Skills, Projects, Certifications, Languages
- Checkbox-based selection for each entry
- Local data persistence using SharedPreferences
- PDF preview and printing functionality
- Professional, ATS-friendly resume template

### Getting Started
Navigate to the `resume_builder` directory and run:
```bash
flutter pub get
flutter run
```

For more details, see [resume_builder/README.md](resume_builder/README.md)
